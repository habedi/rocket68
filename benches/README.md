## Benchmarks

This directory contains benchmarks for comparing Rocket 68 with other emulators.
The benchmarks can be run with the following command:

```bash
make bench
```

> [!NOTE]
> Benchmarks compare Rocket 68 with [Musashi](https://github.com/kstenerud/Musashi)
> and [Moira](https://github.com/dirkwhoffmann/Moira).
